# Results

This directory contains reference CSV outputs from the submitted benchmark package and generated outputs from rerunning scripts.

Typical generated files include:

- `kwnn_benchmark.csv`
- `ensemble_benchmark.csv`
- `mlp_benchmark.csv`
- `transformer_benchmark.csv`
- `latency_benchmark.csv`
- `bootstrap_ci.csv`
- `all_methods_summary.csv`

Generated outputs may vary slightly across platforms for stochastic models and latency measurements. Keep a separate archival copy when preserving exact submission outputs.
